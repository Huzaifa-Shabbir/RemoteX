# Remote_Desktop_Controller_Mobile

**Overview**

RemoteX is a remote desktop control system. The system enables the user to interact with the desktop computer remotely using a mobile device. The mobile app acts as the user interface to interact with the desktop computer remotely.
This repository includes the frontend and backend communication layer of the mobile app, built with Flutter and Dart. The app interacts with the desktop system built with WebRTC and C++.

**Features**

1. Connect mobile device to a desktop system over a network
2. Send control signals from mobile device to the desktop
3. Real-time communication between mobile device and desktop
4. Handling of secure connections
5. Cross-platform mobile interface created with Flutter

**Technology Stack**
Frontend - Mobile App
    Flutter
    Dart
